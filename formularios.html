Elementos de los formularios

Lista de elementos

Nro Nombre del elemento Descripcion Ejemplo
1 input Es un campo de entrada que por medio de su atributo type se puede estipular que tipo de info .requerimos
2 textarea Es un campo de etrada que puede tener multilinea y se define dimensiones mediante las propiedades de cols y rows
3 select
  • Es un elemento que contiene una lista de opciones a elegir
  • Cada opcion esta cpntenida dentro de otra etiqueta llamada option
4 button Elemento que representa a un boton, tiene las mismas caracteristicas que un input:button pero el button tiene etiqueta de apertura y cierre
5 label Es un elemento en el cial se recomienda poner el texto a mostrar para que la pagina sea indexada y mediante su propiedad for al hacer click en el label se seleccionara su input
6 fieldset Es un elemetno en el cual se agrupa otros elementos

7 legend Representa al titulo de algunas etiqyetas por lo general va de la mano con el fieldset

Lista de tipos de input

Numero Tipo Descripcion Ejemplo
1 text Sirve para ingresar texto plano , permite el ingreso de todo tipo de carcateres
2 email Sirve para ingresar correos electroniccos pero a pesar de ser texto plano , mediante la opcion de validar el formilarionse asegura que lo ignresado tenga una @ y un .
3 number Sirve para ingresar solamente numeros enteros y decimales
4 date Sirve para ingresar unicamente fechas , con un calendriode muestra
5 password Sirve para ingresar las contraseñas , oculta lo ingresado por medio de simbolos
6 time Sirve para ingressar las horas y minuto , mas no la fecha
7 color Input para seleccionar un color en base a una paleta de colores
8 checkbox Sirve para seleccionar o deseleccionar un grupo
9 radio Sirve para seleccion de un conjuntod e opciones , solo se puede seleccione uno a la vez mediante su propiedad name
10 file Sirve para subir un archivo local al servidor
11 reset Sirve para limpiar todo el formulario , limpia los campos
12 submit Sirve para enviar todos los campos del formulario , por defecto hara que se recargue la págia(Ejemplo aplicado en la fila 11)
